# Parity Reference Alias

This directory is the normalized migration alias for parity-specific archived evidence.

Current parity artifacts are still stored alongside the archived language roots, including files such as `references/python/PARITY.md` and `references/rust/PARITY.md`.

This alias exists so migration docs and tests can refer to `references/parity/` as the stable parity evidence boundary while archive normalization is still in progress.
