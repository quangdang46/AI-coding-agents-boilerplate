# __PROJECT_NAME__

This project was scaffolded from the AICD TypeScript coding-agent boilerplate.

## Quick start
- install dependencies with your preferred package manager
- edit `boilerplate.config.ts`
- review `__BRAND_DOC__`
- customize `__BRAND_CONFIG__` and `__BRAND_ROOT__/settings.json`
- keep project-local instructions in `__BRAND_ROOT__/instructions.md`
- customize local agents under `__BRAND_ROOT__/agents/`
- customize local skills under `__BRAND_ROOT__/skills/`
- persisted local sessions live under `__BRAND_ROOT__/sessions/`
- `.agents/skills/` remains available as a generic interoperability mirror
- subagents are Markdown files with YAML frontmatter under `__BRAND_ROOT__/agents/`
- add project code under `src/`
