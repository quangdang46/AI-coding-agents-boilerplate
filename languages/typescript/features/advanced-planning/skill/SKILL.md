---
name: advanced-plan
description: Use the optional advanced planning feature-pack workflow for TypeScript generated projects.
whenToUse: Use when the task benefits from explicit planning mode, todo tracking, tool discovery, or skill-guided execution.
allowed-tools: [enter_plan_mode, exit_plan_mode, todo_write, tool_search, skill, file_read]
argument-hint: <goal>
---

1. Enter planning mode before making multi-step implementation decisions.
2. Record the concrete plan with todo tracking before execution.
3. Use tool search or skill lookup only when they materially improve the plan.
4. Exit planning mode once the task is ready to execute.
