# __PROJECT_NAME__

This project was scaffolded from the AICD Rust coding-agent boilerplate.

## Quick start
- review `CLAW.md`
- customize `.claw.json` and `.claw/settings.local.json` for your environment
- customize workspace instructions under `.agent/prompts/`
- customize layered prompt sections under `.agent/prompts/sections/`
- add workspace context fragments under `.agent/context/`
- customize local agents under `.agent/agents/`
- review local skills under `.agent/skills/`
- persisted local sessions can live under `.agent/sessions/`
- local usage records can live under `.agent/usage/`
- add project code under `src/`
- run `cargo fmt` and `cargo test`
