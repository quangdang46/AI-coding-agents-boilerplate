# __PROJECT_NAME__

This project was scaffolded from the AICD Rust coding-agent boilerplate.

## Quick start
- review `__BRAND_DOC__`
- customize `__BRAND_CONFIG__`, `__BRAND_ROOT__/settings.json`, and `__BRAND_ROOT__/settings.local.json`
- keep project-local instructions under `__BRAND_ROOT__/`
- customize local agents under `__BRAND_ROOT__/agents/`
- review local skills under `__BRAND_ROOT__/skills/`
- persisted local sessions live under `__BRAND_ROOT__/sessions/`
- `.agents/skills/` remains available as a generic interoperability mirror
- subagents are Markdown files with YAML frontmatter under `__BRAND_ROOT__/agents/`
- add project code under `src/`
- run `cargo fmt` and `cargo test`
